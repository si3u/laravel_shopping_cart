#Магазин Картин/Модульных картин

#####  Установка

- git clone https://github.com/DmitrySkibitsky/laravel_shopping_cart
- composer install
- Изменить настройки в файле /.env
- Выставить права 755 на /
- Выставить права 777 на: /public/assets/images, /storage 
- php artisan migrate
- php artisan db:seed
